## 모바일 기준 디바이스

iPhone SE (375×667) 기준으로 디자인한다. 이 화면에서 잘리거나 깨지면 안 된다.

## 폰트 크기 원칙

- **모바일 최소 폰트: 15px**. 모든 텍스트는 모바일(375px)에서 15px 이상이어야 한다.
- **PC 최소 폰트: 18px**. 모든 텍스트는 PC(769px 이상)에서 18px 이상이어야 한다. PC 화면은 넓어서 여백이 많고 글씨가 상대적으로 작아 보인다.
- **구현**: `clamp(모바일px, calc(기준px + Xvw), PC_px)` 패턴 사용. `calc()`로 선형 보간하여 모바일~PC 전 구간에서 부드럽게 스케일링.
- **계산 공식**:
  - `vw계수 = (PC_px - 모바일px) / (maxViewport - minViewport) × 100`
  - `기준px = 모바일px - (vw계수 / 100 × minViewport)`
  - 기본 뷰포트: minViewport = 375px (iPhone SE), maxViewport = 1440px
- **예시**: `clamp(15px, calc(12.4px + 0.7vw), 18px)` — 375px→15px, 1440px→18px 선형 보간.
- **왜 calc()가 필요한가**: `clamp(15px, 1.8vw, 18px)` 처럼 vw만 쓰면 375px에서 1.8vw=6.75px → min 15px 고정. 834px까지 스케일링이 시작되지 않아 fluid 효과 없음.
- **숫자/데이터**: 강조 숫자는 PC 기준 30px 이상.
- 항상 PC·모바일(iPhone SE) 양쪽에서 최종 확인할 것.

## 섹션별 기획

### 섹션 1. Hero — "당신에게 매매 기준이 있나요?"

- **목적**: 모든 페르소나를 포괄하는 공통 질문으로 첫인상 후킹
- **헤드라인**: "당신에게 매매 기준이 있나요?"
- **서브 카피**: "언제 사야 하는지, 언제 팔아야 하는지 — 매번 다른 기준으로 결정하고 있다면"
- **스크롤 유도**: 없음. 자연스럽게 아래 진단 섹션으로 이어짐
- **연결 전략**: 서브 카피의 "~다면"이 미완결 문장이라 "그러면 뭐?" 라는 궁금증을 유발. 섹션 1은 질문만 던지고 답은 섹션 2가 담당. 섹션 1에서 결과를 미리 언급하면 섹션 2의 임팩트를 희석시키므로, 의도적으로 비워둔다.

---

### 섹션 2. 진단 — "이 중에 나와 같은 경험이 있나요?"

- **목적**: "내 얘기다" 공감. 고통을 구체적으로 명명해 준다.
- **타겟 페르소나**: 기준 없는 매매자 (고점 매수자, 하락 추세 입자, 늦은 청산자, 뉴스 피로 주린이), 데이터 검증자 (근거 요구자). 조기 익절자는 섹션 3에서 커버; 방향 탐색자·퀀트 입문자·MDD 경시자는 섹션 4·5에서 커버.
- **레이아웃**: 정사각형 카드 6개. PC: 3열×2행 (max-width 640px), 모바일: 2열×3행. 카드 안에 이모지 아이콘 + 경험 문구. 텍스트 18px, 아이콘 36px.

카드 구성:

| 카드 | 타겟 |
|------|------|
| "뉴스 보고 샀다가 고점에 물렸다" | 고점 매수자 |
| "싸 보여서 샀는데, 더 떨어졌다" | 하락 추세 진입자 |
| "더 오를 줄 알았는데, 가지고 있다 보니 떨어졌다" | 늦은 청산자 |
| "버티면 오르겠지 했는데, 결국 손실만 커졌다" | 늦은 청산자 |
| "남들 따라 샀다가 손해만 봤다" | 근거 요구자 |
| "뉴스 볼 때마다 불안해서 자꾸 손이 간다" | 뉴스 피로 주린이 |

- **하단 결론**: "이 패턴의 공통점은 기준이 없었다는 것"
- **전환 다리**: "그렇다면, 기준이 있었다면 어떻게 달랐을까?"

### 섹션 3. 해결책 — "200일선이 기준이 된다"

- **목적**: 기준이 없는 개인투자자에게 200일선이 꽤나 좋은 기준이 될 수 있다는 것을 직관적으로 전달. 이것이 해결책이다. 기술적 상세(앗추 필터 16/20 규칙)나 증명(백테스트 수치)은 하지 않는다 — 그건 섹션 4의 역할.
- **타겟 페르소나**: 기준 없는 매매자 (고점 매수자, 하락 추세 진입자, 조기 익절자, 늦은 청산자)
- **섹션 2→3 연결**: 섹션 2에서 "기준이 없었다"를 깨달은 사용자가 "그러면 기준이 뭔데?"를 품고 내려옴. 여기서 그 답을 준다.

**핵심 메시지:**
- "간단하지만 강력한 기준, 200일선" — 전문가도 쓰는 가장 단순한 기준이 개인투자자에게도 좋은 매매 기준이 된다
- "언제 들어가는가" — 200일선 위에 있으면 들어간다
- "언제 나오는가" — 200일선 아래로 내려오면 나온다
- "투자 전문가라면 기본적으로 사용하는 매매 기준입니다"

**시각화:**
- SPY 실제 차트: **단순 200일선 교차** 기준 진입/청산 신호 표시 (`simpleCrossings`)
  - 노이즈 있는 단순 교차 신호가 그대로 보여야 함 — 섹션 5에서 "이 문제를 앗추 필터가 줄인다"는 서사의 근거
- 보유 구간(초록)과 현금 구간(빨강) 색상 구분
- 1,000만원 훅 및 거래 내역 테이블도 `simpleCrossings` 기반 (단순 200MA 전략 실적)
- **거래 테이블 정렬: 최신 거래가 위** (내림차순)

> **데이터 소스**: `landing_data.json`의 `simpleCrossings` 필드.
> `crossings` (앗추 16/20 필터)는 섹션 5 전용.

- **전환** (두 줄):
  - "최근 몇 년만 잘된 거 아냐?"
  - "SPY만 결과가 좋은 거 아니야?"

### 섹션 4. 신뢰 — "직접 확인해 보셔도 좋습니다"

- **목적**: 섹션 3에서 소개한 200일선 기준이 실제로 작동하는지 데이터로 증명.
- **타겟 페르소나**: 기준 없는 매매자 (뉴스 피로 주린이), 데이터 검증자 (근거 요구자, 퀀트 입문자, MDD 경시자)
- **섹션 3→4 연결**: 섹션 3에서 "최근 몇 년만 잘된 거 아냐?", "SPY만 결과가 좋은 거 아니야?"를 품고 내려온 사용자에게 데이터로 답한다.

**"200일선 매매는 앗추만의 주장이 아닙니다" — 별도 강조 박스(trust-authority-box)로 분리 표시**

**백테스트 — 수익률은 비슷, 리스크만 절반:**
- SPY 전체 기간 기준, 200일선 단순 교차 매매
- 모든 수치는 `backtestComparison.SPY` 계산값 사용 (하드코딩 금지)
  - Buy & Hold: CAGR ≈ 10.5%, MDD ≈ -55.2%
  - 200일선: CAGR ≈ 8.2%, MDD ≈ -28%
- 체감 보조: "1억 투자 시 최대 손실: BnH MDD × 1억 → MA200 MDD × 1억" (동적 계산)
- 상세 거래 이력은 추세 조회 페이지(`/trend_list/SPY`)에서 확인 가능

**전문가 인용 (신뢰 카드):**
- 카드 구조: 인물 이름이 먼저(크게), 인용문은 한국어만 표시, 영어 원문은 아래에 작게
- Paul Tudor Jones: "나의 모든 기준은 200일 이동평균선입니다" (영어 원문: "My metric for everything is the 200-day moving average")
- Ed Seykota: "추세가 당신의 친구입니다" (영어 원문: "The trend is your friend")
- Mebane Faber 논문: 100년 데이터로 검증, SSRN에서 20만회 이상 다운로드

**리딩방 vs 200일선 비교 카드 (좌우 2열):**

| 리딩방 | 200일선 |
|--------|---------|
| 왜 사라고 했는지 모른다 | 200일선 위면 사고, 아래면 판다 |
| 근거를 알 수 없다 | 누구나 직접 확인 가능 |

**마무리 카피:** "틀렸다고 생각하시면 직접 확인하시면 됩니다. 아무 차트에서 200일선을 보시면 됩니다."

- **전환**: "앗추는 여기서 한 단계 더 갔습니다"

### 섹션 5. 앗추 필터 — "거짓 신호를 걸러냅니다"

- **목적**: 200일선의 신뢰가 쌓인 상태에서, 앗추가 단순 200일선 매매의 약점(잦은 매매, 거짓 신호)을 줄여준다는 부가가치 제시.
- **타겟 페르소나**: 공통
- **섹션 4→5 연결**: 섹션 4에서 200일선이 신뢰할 만한 기준임을 확인한 사용자가 "그러면 앗추는 뭐가 다른데?"를 품고 내려옴.

**핵심 메시지:**
- 200일선은 좋은 기준이지만, 선을 넘을 때마다 사고팔면 매매가 잦아지고 거짓 신호에 당한다.
- 앗추는 16/20 규칙으로 이 문제를 줄였다: 최근 20거래일 중 16일 이상 200일선 위일 때만 진입 신호. 단기 변동에 흔들리지 않음.

**시각화:**
- 같은 SPY 차트에서 200일선 단순 매매 신호 vs 앗추 필터 신호를 비교. 거짓 신호가 줄어드는 것이 시각적으로 보여야 함.

**백테스트 비교:**

> **데이터 소스**: 하드코딩 아님. `generate_landing_data.mjs`가 CSV에서 자동 계산.
> 파이프라인 실행 때마다 `landing_data.json`의 `backtestComparison` 필드로 갱신.
> `generate_summary_snapshot.mjs`와 동일한 교차 로직 사용 (수치 정합성 보장).
>
> ```json
> // landing_data.json → backtestComparison.SPY 구조
> {
>   "years": 33.12,
>   "dataStart": "1993-01-29",
>   "dataEnd": "2026-03-13",
>   "ma200": { "cagr": 8.45, "totalSignals": 206, "signalsPerYear": 6.22 },
>   "atchuFilter": { "cagr": 9.11, "totalSignals": 47, "signalsPerYear": 1.42 },
>   "tradeReductionPercent": 77
> }
> ```

**"200일선 대신 앗추 필터로 매매했다면?"** — 백테스트 비교 영역 상단에 질문 표시

**앗추 필터 거래 이력 테이블** (stats 아래): `crossings` (16/20 필터) 기반 2020년 이후 거래 페어링 표시.
- 섹션 3의 단순 200MA 거래 17건과 대비: 앗추 필터 5건의 깔끔함을 이력으로 체감시킴
- 현재 보유 중 포지션은 "(현재)" 표시
- **정렬: 최신 거래가 위** (내림차순) — 현재 포지션 상태가 첫 번째 행에 노출

**카드형 2×2 그리드 레이아웃** (기존 테이블에서 변경). 각 카드는 3×4 내부 그리드 (라벨/200일선/앗추필터 × 수익률/매매횟수/MDD):

| ETF | 200일선 CAGR | 앗추 필터 CAGR | 200일선 매매 | 앗추 매매 | 매매 감소 |
|-----|-------------|---------------|------------|---------|---------|
| SPY | 8.45% | 9.11% | 연 6.2회 | 연 1.4회 | −77% |
| QQQ | 7.69% | 7.38% | 연 7.0회 | 연 1.9회 | −73% |
| XLE | 6.12% | 6.83% | 연 8.2회 | 연 2.0회 | −76% |
| GLD | 7.58% | 9.09% | 연 7.4회 | 연 1.7회 | −77% |

- 수익률은 종목에 따라 높아지기도, 낮아지기도 한다 — 차이는 크지 않다
- 핵심은 **매매 횟수가 연 2회 이내로 감소**한다는 것. 바쁜 직장인도 충분히 가능한 전략이 된다

- **전환**: "SPY가 약해도 기회는 있다"

### 섹션 6. 기회 탐색 — "SPY가 약해도 기회는 있다 — 다만 다른 곳에"

- **목적**: 횡보장/하락장에서도 앗추가 유효함을 데이터로 증명 + 추세 조회 유도
- **타겟 페르소나**: 방향 탐색자 (시장 흐름 미파악자, 시장 회피자), 데이터 검증자 (퀀트 입문자)

**핵심 근거 (데이터):**

- **O'Neil 3/4 법칙**: 시장의 75%는 지수 방향을 따르지만, 25%는 독립적 추세를 유지한다 (William O'Neil, *How to Make Money in Stocks*, IBD 연구)
- **2022년 실증**: QQQ -33% vs XLE(에너지 ETF) +64% — 같은 해, 같은 미국 시장에서 97%p 격차
- **국가 모멘텀 (1970–2024, 54년)**: MSCI World 대비 연 +2.75%p 초과수익, 약 70% 승률 (Faber, SSRN 2010)
- **슈퍼 사이클**: 지수 약세 시대에도 특정 국가·섹터는 10–35년 장기 강세 가능. 2000년대 중국 WTO 후 자원 수출국(호주, 브라질) 강세 / 2020년대 AI 반도체 → 한국(HBM), 대만(TSMC)

**메시지 구조:**

1. "SPY가 약한 해에도 에너지 섹터는 +64% 올랐다" → 섹터 ETF 기회
2. "미국 지수와 상관없이 특정 국가는 강세장이 따로 있다" → 국가 ETF 기회
3. "앗추는 지금 85개 ETF 중 어디에 추세가 살아있는지 보여준다" → 서비스 연결

**표시할 자산 예시 (CAGR 카드):**
- 섹터 ETF: XLE(에너지), XLV(헬스케어), XLI(산업재)
