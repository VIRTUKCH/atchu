# 개별주 추세 조회 — /_stocks

## 이 페이지의 역할

ETF 추세 조회(`/index_etf`)의 개별주 버전. S&P 500 전체 종목의 추세 상태를 리스트로 확인한다.

**답하는 질문:**
- "어떤 종목이 앗추 필터를 통과하고 있나?"
- "기술 섹터 중 반도체는 어떤가?"
- "이격률이 높은/낮은 종목은?"

---

## 유입 경로

- DevPage 허브 → "개별주 추세 조회" 카드 클릭
- `/_stocks_overview` 히트맵에서 종목 클릭 후 뒤로가기
- `/_stocks_overview` 섹터 추세 강도 카드 클릭 → `/_stocks?sector=기술`
- Discord 관리자 채널 알림 하단 링크

---

## 페이지 구조

| 순서 | 영역 | 설명 |
|------|------|------|
| 1 | 요약 바 | "S&P 500 {N}개 중 {M}개 종목이 앗추 필터 통과" |
| 2 | 섹터 필터 | GICS 11개 섹터 칩 (전체 / 기술 / 헬스케어 / ...) |
| 3 | 서브섹터 필터 | 섹터 선택 시 해당 섹터의 서브섹터 칩 추가 표시 |
| 4 | 검색 | 티커명 또는 회사명 (예: AAPL, Apple) |
| 5 | 정렬 | 섹터별 / 이격률 높은순·낮은순 / 수익률 높은순 / MDD 낮은순 |
| 6 | 카드 그리드 | EtfSummaryCard 재사용. 클릭 → `/_stocks/:ticker` |

---

## 필터링 동작

### 섹터 필터 (1단계)

- TypeFilter 컴포넌트 재사용
- GICS 11개: 기술, 헬스케어, 금융, 산업재, 임의소비재, 필수소비재, 통신, 에너지, 유틸리티, 부동산, 소재
- "전체" 선택 시 503종목 모두 표시
- URL: `/_stocks?sector=기술`

### 서브섹터 필터 (2단계)

- 섹터 선택 시 해당 섹터의 서브섹터 칩이 아래에 추가 표시
- TypeFilter 컴포넌트를 두 번째 줄에 재사용
- 서브섹터가 2개 이상일 때만 표시 (1개면 의미 없음)
- "전체" 선택 시 해당 섹터 전체 표시
- URL: `/_stocks?sector=기술&sub=반도체` (예정)

**예시 — 기술 선택 시:**
```
[전체] [기술] 헬스케어  금융  산업재  ...     ← 섹터 필터
[전체] [소프트웨어] [반도체] [하드웨어] [IT서비스]  ← 서브섹터 필터
```

### URL 파라미터로 서브섹터 직접 진입

현재: `/_stocks?sector=기술` → 기술 전체
추가: `/_stocks?sector=기술&sub=반도체` → 기술 > 반도체만

**사용 시나리오:**
- `/_stocks_overview`에서 서브섹터 클릭 시 직접 이동
- Discord 알림에서 특정 서브섹터 링크

### 섹터 전환 시 서브섹터 초기화

- 섹터 변경 → 서브섹터 "전체"로 리셋
- "전체(ALL)" 선택 → 서브섹터 필터 숨김

---

## 정렬 옵션

| 값 | 라벨 | 기준 |
|---|---|---|
| `sector` | 섹터별 | SECTOR_ORDER 순서 → 알파벳 |
| `ma200_desc` | 이격률 높은순 | 200일선 대비 이격률 내림차순 |
| `ma200_asc` | 이격률 낮은순 | 200일선 대비 이격률 오름차순 |
| `cagr_desc` | 수익률 높은순 | 최적 CAGR 내림차순 |
| `mdd_asc` | MDD 낮은순 | 최적 기간 MDD 오름차순 (절대값 작은 순) |

---

## ETF 추세 조회와의 차이

| | ETF (`/index_etf`) | 개별주 (`/_stocks`) |
|---|---|---|
| 종목 수 | 85개 | ~503개 |
| 분류 기준 | 자산군 (10개) | GICS 섹터 (11개) + 서브섹터 (46개) |
| 서브 필터 | 없음 | 서브섹터 필터 칩 |
| CSV 로딩 | 빌드 시 번들 | on-demand fetch |
| 타겟 | 일반 사용자 | 개발자 전용 |
| 네비게이션 | 상단 탭 노출 | 숨김 |

## 관련 컴포넌트

- `StockListPage.jsx` — 페이지 본체
- `TypeFilter.jsx` — 섹터/서브섹터 필터 칩 (재사용)
- `EtfSummaryCard.jsx` — 카드 (ETF와 동일 컴포넌트 재사용)
- `stockDataLoaders.js` — 데이터 로더 (`stockTickerMetaMap`, `stockSnapshotMap`)

## 관련 데이터

- `sp500.json` — 티커 메타 (`type`, `subType`, `short_description`)
- `stock_snapshots.json` — 스냅샷 (가격, 이격률, 앗추 필터 상태, CAGR, MDD)
