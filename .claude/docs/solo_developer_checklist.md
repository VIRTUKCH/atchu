## 2. 법적 리스크 → [law_risk/](./law_risk/)

겸업, 금융규제, 데이터 라이선스 — **앗추에 가장 중요한 리스크 영역**.

| 문서 | 내용 |
|------|------|
| [00_legal_risk_overview.md](./law_risk/00_legal_risk_overview.md) | 자본시장법, 금소법, 개인정보보호법, EODHD 라이선스 종합 분석 |

### 핵심 요약

**겸업 리스크** — 🔴 즉시 확인
- 근로계약서 겸직/IP/NDA 조항 확인 필수
- 회사가 핀테크/금융 관련이면 경쟁사업 해석 가능
- 회사 장비·시간·리소스 절대 사용 금지
- Git 커밋 시간 주의 (근무시간 중 커밋 = 불리한 증거)

**금융 규제** — 🔴 즉시
- 무료 시: 불특정 다수 대상 예외 적용 가능
- 유료화 시: **투자자문업 등록 필수** (미등록 → 3년 징역 / 1억 벌금)
- "매수/매도 신호" → "기술적 지표 변화" 표현 순화
- 모든 수익률 표시에 면책 문구 필수

**데이터 라이선스** — 🟡 단기
- EODHD에 현재 사용 방식 허용 여부 이메일 확인
- 사이트 푸터에 데이터 출처 명시
- 유료화 전 상업 라이선스 확보 또는 대안 검토
