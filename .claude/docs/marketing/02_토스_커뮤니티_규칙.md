# 토스증권 커뮤니티서비스 이용규칙

> 최초 개시일: 2021-05-20 / 최종 수정일: 2026-02-24

## 커뮤니티 운영 기조
보수적이고 선제적 기조. 위반 소지만 있어도 제재 가능.

---

## 게시 제한 사항

### 불건전 콘텐츠
- 욕설, 비방, 모욕, 명예훼손, 외설 (은어·속어·초성 등 표현방식 불문)
- 출처 불분명한 허위 정보 또는 링크
- 이용자 오인·혼동 유발 / 건전한 투자 판단 저해하는 정보·링크
- 무분별한 반복 게재 (도배)
- 주제와 연관 없는 글·댓글
- 투자와 무관한 정치 논쟁

### 홍보·광고 (핵심 주의)
- **투자와 무관한 상품/서비스 홍보·광고 금지**
- **홍보성·광고 목적 글·댓글 금지**
- 영리 목적, 투자 권유 목적 금지
- 연락창구·초대링크(전화번호, 이메일, 채팅방 링크 등) 기재 금지
- (유사)투자자문업, 집합투자업 홍보 금지

### 기타
- 명의도용, 저작권 침해
- 개인정보 노출
- 시장 질서 교란 행위 (허위 정보 유포, 시세조종 관련)

---

## 제재 구조

### 경고 누적 → 이용 제한
| 누적 경고 | 제한 기간 |
|-----------|-----------|
| 1회 | 1시간 |
| 2~5회 | 1일 (24시간) |
| 6회 | 2일 |
| 7회 | 3일 |
| 8회 | 4일 |
| 9회 | 5일 |
| 10회 | **영구 제한** |

- 영구 제한 시 작성한 모든 글 삭제 가능
- 탈퇴 후 재가입해도 제한 기간 유지됨

### 즉시 제재 (경고 누적 없이)
- 도배, 리딩방 홍보, 음란게시물
- 범죄 정황, 개인정보 유출, 해킹

### 모니터링
- AI 기반 + 사람 모니터링 병행
- 신고를 받아도 토스 자체 판단으로 삭제 안 할 수 있음 (역으로, 신고 없어도 삭제 가능)

## 앗추 마케팅 시 유의점

| 행위 | 위험도 | 비고 |
|------|--------|------|
| 앗추 링크 직접 게재 | 높음 | "건전한 투자 판단 저해 링크" 또는 "홍보성 글"로 판단 가능 |
| "앗추에서 확인하세요" 언급 | 중간 | 맥락에 따라 홍보성 판단 가능 |
| 추세추종 개념 자연스럽게 공유 | 낮음 | 종목 토론과 연관성 있을 때 |
| 반복 댓글 | 높음 | 도배로 즉시 영구 제한 |
