# 앗추 마케팅 전략 총괄

> 최종 수정: 2026-04-10 (채널 전략 종목토론방 단일 집중으로 간소화)

## 1. 현황 진단

### 서비스 요약
- **서비스**: 앗추 — 섹터·국가·채권·원자재·중소형주 ETF 추세추종 데이터 서비스 (146개 자산)
- **타겟**: 한국 개인 투자자 (뉴스에 휘둘리는 매매 패턴을 개선하고 싶은 사람)
- **배포**: Vercel 정적 사이트 + Discord 알림
- **수익 모델**: 2단계 — Phase 1 무료(유입·리텐션) → Phase 2 유료(디스코드 1000명+ 시점)
- **현재 단계**: Phase 1 — 종목토론방으로 사람을 데려와서 디스코드로 유입시키는 것이 최우선

### 제약 조건

| 항목 | 내용 |
|------|------|
| **시간** | 본업 개발자(하루 9시간 근무). 마케팅 투입 시간 극히 제한적. |
| **예산** | 월 10만원 이하. 과금 마케팅 불가 → 무과금 or 최소 비용 채널 중심 |
| **운영자** | 1인 운영 (기획·개발·마케팅 모두 본인) |

### 현재 마케팅 인프라 상태

| 항목 | 상태 | 비고 |
|------|------|------|
| 랜딩 페이지 | ✅ 완성 | 6개 섹션, 다크/라이트 모드 |
| Discord 채널 | ✅ 운영 중 | 일간 리포트 + 추세 알림 자동 발송 |
| 이용약관 | ✅ 완성 | |

| OG 이미지 | ✅ 있음 | chaeho.store 도메인 하드코딩 |

| robots.txt | ❌ 없음 | |

| EODHD 데이터 출처 | ⚠️ 미표기 | 사이트 푸터에 명시 필요 |

### 핵심 문제
> **마케팅 효과를 측정할 수단이 전혀 없다.**
> 어떤 채널에서 유입이 오는지, CTA 클릭률이 얼마인지, Discord 전환이 몇 건인지 알 수 없음.

### 1-1. 성장 전략 — 2단계 접근

> **당장 결제는 비현실적. 먼저 사람을 데려와야 한다.**

#### Phase 1: 유입 & 충성 확보 (무료, 디스코드 1000명까지)

**핵심 퍼널**:
```
[즉시] 종목토론방 (정보성 참여) → 사이트 방문 → 디스코드 유입 → 매일 알림으로 습관화
```

**리텐션 요소** — 사용자가 매일 돌아오게 만드는 것:

| 요소 | 역할 | 현재 |
|------|------|------|
| 이평선 크로스 알림 | 핵심 — "오늘 내 종목 뭐 됐지?" | ✅ 디스코드 알림 있음 |
| 종목별 토론방 | 소속감 — "나만 보는 게 아니구나" | ❌ 없음 |
| 주간/월간 리포트 | "이 서비스 쓰니까 판단이 나아졌다" | ❌ 없음 |

**마일스톤**:
```
디스코드 100명  → 컨셉 검증됨, 계속 갈 가치 있음
디스코드 300명  → 종목별 토론방 활성화 시작
디스코드 500명  → 앱 개발 착수 타이밍
디스코드 1000명 → 유료화 시작
```

**성공 지표**: DAU/MAU 비율 30% 이상 (매일 습관적으로 확인하는 수준)

#### Phase 2: 수익화 (디스코드 1000명+)

**필수 선행**: 유사투자자문업 등록 (초기비용 50~100만원)

| 무료 (유지) | 유료 (추가) |
|-------------|-------------|
| 일일 알림 (디스코드) | 앱 푸시 알림 (실시간) |
| 기본 추세 현황 (웹) | 상세 보고서 + 백테스트 결과 |
| 146종목 이평선 데이터 | 커스텀 알림 (내 종목, 내 조건) |

| 레벨 | 의미 | 채널 |

### Day 0: 즉시 시작 — 종목토론방 게릴라 마케팅

> **인프라·파이프라인 구축 불필요. 지금 당장 시작 가능한 유일한 채널.**

- **채널**: 네이버 종토방 / 토스증권 종토방
- **닉네임**: 앗추
- **방식**: 정보성 시장 관찰 글 주 2~3회 → 물어볼 때만 "앗추 검색해보세요"
- **비용**: $0
- 상세: [`01_종목토론방_게릴라마케팅.md`](01_종목토론방_게릴라마케팅.md)

---

### Phase 0: 기반 인프라 (1회성)

#### 트래킹 도구 설치
- **자동화 레벨**: C (1회성)
- **방법**: Vercel Analytics (무료, 코드 1줄) 또는 Google Analytics 4
- **측정 항목**: 페이지뷰, CTA 클릭, Discord 링크 클릭, 유입 경로(UTM)
- **난이도**: 30분
- **비용**: $0

#### SEO 메타태그 정비
- **자동화 레벨**: C (1회성)
- **작업**: meta description 추가, og:description 카피 업데이트, og:url, canonical, robots.txt, sitemap.xml
- **난이도**: 반나절
- **비용**: $0

#### Programmatic SEO (종목별 랜딩 페이지)
- **자동화 레벨**: C (1회성 구축, 종목 추가 시 자동 확장)
- **방법**: react-helmet으로 종목별 동적 title/description/OG 생성 + sitemap.xml에 146개 URL 자동 등록
- **난이도**: 1~2일
- **비용**: $0
- **기대 효과**: 146개 종목 × 롱테일 키워드 조합 ("SPY 200일선 분석", "QQQ 추세추종 전략" 등) → 검색 진입점 확보

---

### Discord 커뮤니티 (운영 중)
- **현재 상태**: 일간 리포트 + 추세 알림 자동 발송 완료
- **확장**: 사용자 토론 채널 추가, "오늘의 시장" 토론 유도
- **역할**: 핵심 리텐션 채널
- **⚠️ 법적 주의 (유사투자자문업)**:
  - 알림 전용 채널은 **채팅 불가(일방향)로 설정** 필수 — 양방향 채팅 가능 시 "리딩방" 규제 대상 가능
  - 토론 채널에서 운영자가 특정 종목 매수/매도 의견을 제시하면 안 됨
  - 유료 멤버십 전환 시 **투자자문업 등록 필수** (미등록 시 3년 이하 징역 또는 1억원 이하 벌금)

## 7. 핵심 원칙

1. **측정 먼저**: Analytics 없이 마케팅은 의미 없다. 측정 가능하게 만들고 나서 트래픽을 태운다.
2. **데이터가 소재**: 매일 생성되는 추세 데이터 자체가 마케팅 소재다. 별도 콘텐츠 기획 불필요.
3. **법적 안전 우선**: 모든 콘텐츠에 면책 문구 포함. "신호/전략" 대신 "감지/지표" 용어 사용. 유료화 전 법률 검토 필수.
4. **유입 최우선**: 결제 전에 사람을 데려오는 것이 먼저. 디스코드 1000명이 유료화 시작점.
