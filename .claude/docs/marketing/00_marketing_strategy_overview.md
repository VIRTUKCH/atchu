## 1. 현황 진단

### 서비스 요약
- **서비스**: 앗추 — 섹터·국가·채권·원자재·중소형주 ETF 추세추종 데이터 서비스 (100개 이상의 ETF)
- **타겟**: 한국 개인 투자자 (뉴스에 휘둘리는 매매 패턴을 개선하고 싶은 사람)
- **배포**: Vercel 정적 사이트 + Discord 알림
- **수익 모델**: 3단계 — 종목토론방 → AI 영상 콘텐츠 → 앱 개발·유료화
- **현재 단계**: Step 1 — 종목토론방으로 사람을 데려와서 디스코드로 유입

### 제약 조건

| 항목 | 내용 |
|------|------|
| **시간** | 본업 개발자(하루 9시간 근무). 마케팅 투입 시간 극히 제한적. |
| **예산** | 월 10만원 이하. 과금 마케팅 불가 → 무과금 or 최소 비용 채널 중심 |
| **운영자** | 1인 운영 (기획·개발·마케팅 모두 본인) |

### 인프라 현황
> ✅ 마케팅 인프라 준비 완료. GA4 운영 중 (G-S4SR7KDWZR), 면책 조항은 유료화 전 법률 검토 시점에 강화 예정.

## 2. 성장 전략 — 3단계

> **당장 유료화는 비현실적. 먼저 사람을 데려와야 한다.**

### Step 1: 종목토론방 (지금 당장)
- 네이버/토스증권 종토방에서 정보성 참여로 자연 유입
- 상세: [`01_종목토론방_게릴라마케팅.md`](01_종목토론방_게릴라마케팅.md)

### Step 2: AI 영상 콘텐츠 (트래픽 확인 후)
- AI로 영상 제작 → 유튜브 Shorts, 인스타 릴스, X, 틱톡 멀티 배포
- 앗추 데이터가 매일 소재 자동 생성 → 콘텐츠 기획 부담 없음

### Step 3: 앱 개발 (사용자 충분히 모이면)
- 웹 → 앱으로 확장, 푸시 알림 등 리텐션 강화
- 유사투자자문업 등록 후 유료화 검토

### 마일스톤

```
디스코드 100명  → 컨셉 검증, Step 2 시작
디스코드 500명  → 앱 개발 착수 타이밍
디스코드 1000명 → 유료화 시작
```

## 3. 채널 전략

### Step 1: 종목토론방
- **채널**: 토스증권 종토방 (네이버 증권 종토방은 제외 — 아래 참고)
- **닉네임**: 이백일선
- **방식**: 정보성 시장 관찰 글 주 2~3회 → 물어볼 때만 "앗추 검색해보세요"
- **비용**: $0
- 상세: [`01_종목토론방_게릴라마케팅.md`](01_종목토론방_게릴라마케팅.md)

#### 네이버 증권 종토방을 쓰지 않는 이유
- **국장 중심 사용자층**: 네이버 증권은 코스피/코스닥 중심 투자자 비중이 높음. 앗추(미국 섹터·국가 ETF 중심)와 타겟 미스매치
- **미장 투자자 비중 불명확**: 네이버 증권 사용자 중 미국 ETF에 관심 있는 사람이 얼마나 되는지 확인할 방법이 없음. 전환율 예측 자체가 어려움
- **활성도**: 언론 보도 기준, 네이버는 MAU가 훨씬 많지만 일 게시물 수는 토스와 비슷한 수준으로 알려짐 (정확한 비교 수치는 공식 확인 어려움)
- **결론**: 실제로 해봐야 알 수 있지만, 다른 채널들이 직관적으로 더 효율적으로 보여서 우선순위에서 제외. 토스 집중 + 다른 채널 탐색이 현재 전략

### Step 1 병행: Programmatic SEO (지금 당장)
- **채널**: 구글 / 네이버 오가닉 검색
- **방식**: 146개 ETF 티커별 자동 페이지 생성 (React 동적 라우팅)
- **타겟 키워드 패턴**: `"QQQ 추세"`, `"SPY 200일선"`, `"섹터 ETF 추세추종"`, `"[티커] 이동평균"` 등
- **구현 비용**: 개발 1~2주 (1회성), 이후 유지 거의 불필요
- **기대 효과**: 구현 후 4~8주 인덱싱, 3~6개월 내 복리 트래픽 시작
- **실증 사례**: Groww(인도 투자앱) pSEO → 월 20M 트래픽 / Failory(1인 개발) → 월 97k 방문 중 54%가 pSEO 유입
- **주의**: 데이터만 있는 빈 페이지는 구글 EEAT 불이익 → 각 티커 페이지에 섹터 설명·ETF 특성 2~3줄 editorial 콘텐츠 추가 필요
- **비용**: $0

### Step 1 병행: 네이버 블로그 SEO (지금 당장)
- **채널**: 네이버 블로그 (검색 점유율 57.86%)
- **방식**: 추세추종·ETF 전문 블로그 → 앗추 링크 자연 삽입
- **추천 콘텐츠**: `"추세추종 ETF 투자란 무엇인가"`, `"200일 이동평균선 활용법"`, `"[티커] 추세 월별 업데이트"`
- **C-Rank 전략**: 한 카테고리(추세추종/ETF) 집중 + 주 1~2편 꾸준히
- **시간 투자**: 주 1~2시간
- **비용**: $0

### Step 1 병행: 네이버 카페 진성 참여 (여유 있을 때)
- **채널**: 미국주식이 미래다 (회원 33만명, 국내 최대 미국주식·ETF 카페)
- **방식**: 직접 홍보 글 금지 → ETF 관련 질문에 앗추 데이터 스크린샷으로 진성 답변
- **비용**: $0

### Step 2: AI 영상 콘텐츠 (Discord 100명 이후)
- **채널**: 유튜브 Shorts, 인스타 릴스, X, 틱톡
- **방식**: AI 영상 제작 툴 활용 → 앗추 데이터 기반 시장 관찰 영상 (Faceless 형태)
- **소재**: 매일 생성되는 추세 데이터 (별도 기획 불필요)
- **시간 투자**: 주 4~5시간 / 월 15편 이상이 성장 임계치
- **비용**: AI 툴 구독료 수준 (저비용)

### Discord 커뮤니티 (운영 중)
- **현재 상태**: 일간 리포트 + 추세 알림 자동 발송 완료
- **역할**: 핵심 리텐션 채널
- **성장 전술 (0 → 100명)**:
  - 종토방·네이버 카페·블로그 글에 Discord 링크 삽입
  - 신호 발송 후 이모지 투표 부착 ("이 신호 보고 어떻게 하셨어요? 👍매수 / 👀관망 / 🤔공부중")
  - 피드백 반영 시 반드시 채널에 공지 (폐루프 — 다음 피드백 유도)
- **성장 전술 (100 → 1,000명)**:
  - Disboard, DiscordMe 리스팅 + 정기 bump
  - 앗추 독점 데이터 알림 (Discord에서만 제공하는 콘텐츠 확보)
  - 활성 멤버 모더레이터 승격 (소속감·책임감 부여)
- **벤치마크**: NVSTly(fintech) — Discord 중심으로 40,000명+, 월 $12,000 달성
- **⚠️ 법적 주의 (유사투자자문업)**:
  - 알림 전용 채널은 **채팅 불가(일방향)로 설정** 필수
  - 토론 채널에서 운영자가 특정 종목 매수/매도 의견 제시 금지
  - 유료 멤버십 전환 시 **투자자문업 등록 필수** (미등록 시 3년 이하 징역 또는 1억원 이하 벌금)

## 10. 핵심 원칙

1. **데이터가 소재**: 매일 생성되는 추세 데이터 자체가 마케팅 소재다. 별도 콘텐츠 기획 불필요.
2. **법적 안전 우선**: 모든 콘텐츠에 면책 문구 포함. "신호/전략" 대신 "감지/지표" 용어 사용.
3. **유입 최우선**: 유료화 전에 사람을 데려오는 것이 먼저. 디스코드 1000명이 유료화 시작점.
