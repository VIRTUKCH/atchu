> 최종 수정: 2026-04-11

---

## 1. 현황 진단

### 서비스 요약
- **서비스**: 앗추 — 섹터·국가·채권·원자재·중소형주 ETF 추세추종 데이터 서비스 (100개 이상의 ETF)
- **타겟**: 한국 개인 투자자 (뉴스에 휘둘리는 매매 패턴을 개선하고 싶은 사람)
- **배포**: Vercel 정적 사이트 + Discord 알림
- **수익 모델**: 3단계 — 종목토론방 → AI 영상 콘텐츠 → 앱 개발·유료화
- **현재 단계**: Step 1 — 종목토론방으로 사람을 데려와서 디스코드로 유입

### 제약 조건

| 항목 | 내용 |
|------|------|
| **시간** | 본업 개발자(하루 9시간 근무). 마케팅 투입 시간 극히 제한적. |
| **예산** | 월 10만원 이하. 과금 마케팅 불가 → 무과금 or 최소 비용 채널 중심 |
| **운영자** | 1인 운영 (기획·개발·마케팅 모두 본인) |

### 인프라 현황
> ✅ 마케팅 인프라 준비 완료. GA4 운영 중 (G-S4SR7KDWZR), 면책 조항은 유료화 전 법률 검토 시점에 강화 예정.

## 2. 성장 전략 — 3단계

> **당장 유료화는 비현실적. 먼저 사람을 데려와야 한다.**

### Step 1: 종목토론방 (지금 당장)
- 네이버/토스증권 종토방에서 정보성 참여로 자연 유입
- 상세: [`01_종목토론방_게릴라마케팅.md`](01_종목토론방_게릴라마케팅.md)

### Step 2: AI 영상 콘텐츠 (트래픽 확인 후)
- AI로 영상 제작 → 유튜브 Shorts, 인스타 릴스, X, 틱톡 멀티 배포
- 앗추 데이터가 매일 소재 자동 생성 → 콘텐츠 기획 부담 없음

### Step 3: 앱 개발 (사용자 충분히 모이면)
- 웹 → 앱으로 확장, 푸시 알림 등 리텐션 강화
- 유사투자자문업 등록 후 유료화 검토

### 마일스톤

```
디스코드 100명  → 컨셉 검증, Step 2 시작
디스코드 500명  → 앱 개발 착수 타이밍
디스코드 1000명 → 유료화 시작
```

## 3. 채널 전략

### Step 1: 종목토론방
- **채널**: 토스증권 종토방
- **닉네임**: 앗추
- **방식**: 정보성 시장 관찰 글 주 2~3회 → 물어볼 때만 "앗추 검색해보세요"
- **비용**: $0
- 상세: [`01_종목토론방_게릴라마케팅.md`](01_종목토론방_게릴라마케팅.md)

### Step 2: AI 영상 콘텐츠
- **채널**: 유튜브 Shorts, 인스타 릴스, X, 틱톡
- **방식**: AI 영상 제작 툴 활용 → 앗추 데이터 기반 시장 관찰 영상
- **소재**: 매일 생성되는 추세 데이터 (별도 기획 불필요)
- **비용**: AI 툴 구독료 수준 (저비용)

### Discord 커뮤니티 (운영 중)
- **현재 상태**: 일간 리포트 + 추세 알림 자동 발송 완료
- **역할**: 핵심 리텐션 채널
- **⚠️ 법적 주의 (유사투자자문업)**:
  - 알림 전용 채널은 **채팅 불가(일방향)로 설정** 필수
  - 토론 채널에서 운영자가 특정 종목 매수/매도 의견 제시 금지
  - 유료 멤버십 전환 시 **투자자문업 등록 필수** (미등록 시 3년 이하 징역 또는 1억원 이하 벌금)

## 7. 법적 컴플라이언스

> 상세: [`../law_risk/00_legal_risk_overview.md`](../law_risk/00_legal_risk_overview.md)

### 마케팅 콘텐츠 용어 규칙

| 금지 표현 | 안전한 대체 표현 | 이유 |
|---------|--------------|------|
| "매수/매도 신호" | "200일선 돌파/이탈 감지" | 투자자문 해석 방지 |
| "추세 진입" (단독) | "200일 추세 진입 감지" | 매수 암시 완화 |
| "장기 상승 흐름에 합류" | "장기 상승 추세 진입 감지" | "합류" = 매수 암시 |
| "앗추 전략" | "앗추 필터" | "전략" = 투자 전략 권유 |
| "구매합니다" / "담습니다" | "신호를 확인합니다" | 직접적 매수 권유 |
| "지금이 기회" / "지금 사야" | 사용 불가 | 긴급감 유도 금지 |
| "보장된 수익" / "확실한 수익" | 사용 불가 | 미래 수익 보장 암시 |
| "뉴스보다 먼저 알 수 있습니다" | "추세 변화를 데이터로 감지합니다" | 우월성 과장 |

### 유료화 전 필수 조치

1. **유사투자자문업 등록** (미등록 시 3년 징역/1억 벌금)
2. **통신판매업 신고** (공정거래위원회)
3. **개인정보 처리방침** 작성
4. **환불/청약철회 정책** 수립
5. **EODHD 상업 라이선스** 확보 또는 대안 데이터 소스
6. **금융법 전문 변호사 상담**

## 8. 핵심 원칙

1. **데이터가 소재**: 매일 생성되는 추세 데이터 자체가 마케팅 소재다. 별도 콘텐츠 기획 불필요.
2. **법적 안전 우선**: 모든 콘텐츠에 면책 문구 포함. "신호/전략" 대신 "감지/지표" 용어 사용.
3. **유입 최우선**: 유료화 전에 사람을 데려오는 것이 먼저. 디스코드 1000명이 유료화 시작점.
