# 세금 및 사업자 등록

마지막 업데이트: 2026-04-06

> **면책**: 이 문서는 일반적인 참고 정보이며, 법률/세무 자문이 아닙니다. 구체적인 사안은 반드시 세무사에게 상담하세요.

---

## 1. 현재 상태

| 항목 | 상태 | 비고 |
|------|------|------|
| 사업자 등록 | 없음 | 수익 없는 무료 서비스 |
| 부가가치세 신고 | 해당 없음 | |
| 종합소득세 신고 | 근로소득만 | 사업소득 없음 |

**현재 수익화 계획 없음** — 아래 내용은 향후 수익 발생 시 참고용.

---

## 2. 사업자 등록이 필요한 시점

| 상황 | 사업자 등록 |
|------|-----------|
| 완전 무료, 수익 없음 | **불필요** (현재 상태) |
| YouTube 광고 수익 발생 | 필요 (사업 개시 20일 이내) |
| 유료 구독 판매 | 필요 |
| 후원/슈퍼챗 수익 | 필요 |

---

## 3. 수익화 시 체크리스트 (참고용)

| # | 항목 | 비고 |
|---|------|------|
| 1 | 사업자 등록 (사업 개시 20일 이내) | 홈택스 온라인 신청 가능 |
| 2 | 통신판매업 신고 | 관할 구청, 등록면허세 약 4만원 |
| 3 | 과세 유형 결정 (간이 vs 일반) | 초기엔 간이과세자 권장 |
| 4 | 부가세 신고 일정 확인 | 간이: 1월 / 일반: 1월·7월 |
| 5 | 종합소득세 합산 신고 (5월) | 근로소득 + 사업소득 합산 |
| 6 | 경비 영수증 관리 | 서버비·소프트웨어 비용 공제 가능 |
| 7 | 세무사 상담 | 유료화 직전 1회 권장 |

### 업종코드 (사업자 등록 시)
- `722000` — 소프트웨어 개발 및 공급업
- `921505` — 미디어 콘텐츠 창작업 (YouTube 중심이면)

### 간이과세자 vs 일반과세자

| 구분 | 간이과세자 | 일반과세자 |
|------|-----------|-----------|
| 조건 | 연 매출 1억 400만원 미만 | 제한 없음 |
| 부가세율 | 업종별 1.5~4% | 10% (매입세액 공제) |
| 신고 주기 | 연 1회 (1월) | 연 2회 (1월, 7월) |
| 세금계산서 | 발행 불가 (매출 4,800만원 미만) | 발행 가능 |
| 주의 | SW 개발 업종은 간이과세 거부 사례 있음 | |
