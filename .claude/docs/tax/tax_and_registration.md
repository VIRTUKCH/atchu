# 세금 및 사업자 등록

마지막 업데이트: 2026-02-22

> **면책**: 이 문서는 일반적인 참고 정보이며, 법률/세무 자문이 아닙니다. 구체적인 사안은 반드시 세무사에게 상담하세요.

---

## 1. 현재 상태

| 항목 | 상태 | 비고 |
|------|------|------|
| 사업자 등록 | 해당 없음 | 현재 무료 서비스 |
| 부가가치세 신고 | 해당 없음 | |
| 종합소득세 신고 | 근로소득만 | 사업소득 없음 |

---

## 2. 사업자 등록이 필요한 시점

| 상황 | 사업자 등록 | 비고 |
|------|-----------|------|
| 완전 무료, 수익 없음 | 불필요 | 현재 상태 |
| YouTube 광고 수익 발생 | **필요** | 사업 개시 20일 이내 |
| 유료 구독 판매 | **필요** | |
| 후원/슈퍼챗 수익 | **필요** | |

---

## 3. 사업자 등록 방법

### 3-1. 절차

1. 관할 세무서 방문 또는 [홈택스](https://www.hometax.go.kr) 온라인 신청
2. 업종코드 선택:
   - `722000` — 소프트웨어 개발 및 공급업
   - `921505` — 미디어 콘텐츠 창작업 (YouTube 중심이면)
3. 과세 유형 결정

### 3-2. 간이과세자 vs 일반과세자

| 구분 | 간이과세자 | 일반과세자 |
|------|-----------|-----------|
| 조건 | 연 매출 1억 400만원 미만 | 제한 없음 |
| 부가세율 | 업종별 1.5~4% | 10% (매입세액 공제) |
| 신고 주기 | 연 1회 (1월) | 연 2회 (1월, 7월) |
| 세금계산서 | 발행 불가 (매출 4,800만원 미만) | 발행 가능 |
| 주의 | SW 개발 업종은 간이과세 거부되는 경우 있음 | |

→ 초기에는 간이과세자로 시작, 매출 커지면 일반과세자 전환.

---

## 4. 세금 신고

### 4-1. 부가가치세

| 과세 유형 | 신고 시기 | 비고 |
|----------|---------|------|
| 간이과세자 | 매년 1월 | 연 1회 |
| 일반과세자 | 1월, 7월 | 연 2회 |

- YouTube 광고수익(Google Adsense)은 **영세율(0%)** 적용 가능 (해외 용역)
- 영세율이라도 **신고 자체는 필수**

### 4-2. 종합소득세

- 신고 시기: 매년 **5월**
- 직장 급여(근로소득) + 부업 수익(사업소득) **합산 신고**
- 미신고 시 최대 **40% 가산세**

### 4-3. YouTube 광고 수익 세금

- Google Adsense 수익 = **사업소득**
- 장비, 소프트웨어 비용, 서버 비용 등 **필요경비 공제** 가능
- 경비 인정을 위해 **영수증/카드 내역 보관 필수**
- Google에서 원천징수하는 미국 세금(30%)은 한미 조세조약으로 환급/감면 가능

---

## 5. 통신판매업 신고

유료 서비스를 온라인으로 판매하면 **통신판매업 신고**가 별도로 필요하다.

| 항목 | 내용 |
|------|------|
| 신고처 | 관할 시/군/구청 |
| 비용 | 등록면허세 약 4만원 |
| 시점 | 유료 판매 개시 전 |
| 미신고 시 | 과태료 부과 |

---
