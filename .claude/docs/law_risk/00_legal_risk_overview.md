## 1. 자본시장법 — 투자 관련 서비스의 법적 분류

### 1-1. 핵심 쟁점

자본시장법 제6조 제7항은 "금융투자상품에 대한 투자판단(종목, 취득·처분의 방법·시기 등)에 관한 자문에 응하는 것을 영업으로 하는 것"을 투자자문업으로 정의한다.

**앗추가 제공하는 것:**
- 추세 진입/이탈 "신호"를 매일 자동 발송
- 특정 종목의 매수/매도 시점을 알려주는 형태

**법적 해석:**
- "매수 신호", "매도 신호" 표현 → **투자판단 자문으로 해석될 가능성 높음**
- Discord 알림에서 "추세 진입 감지"라는 표현을 사용 (기존 "장기 상승 흐름에 합류"에서 변경 완료)

### 1-2. 한국 법상 3가지 분류

투자 관련 서비스는 규제 수준에 따라 3단계로 나뉜다.

#### A. 투자자문업 (가장 무거운 규제)

**정의**: 특정인에게 **개별화된** 투자판단(종목, 매수/매도 시점 등)을 자문하는 영업.

| 항목 | 내용 |
|------|------|
| 방식 | 금융위원회에 **등록** |
| 법인 | **법인만 가능** (개인 불가) |
| 자본금 | **1억~2.5억원** (투자 대상에 따라 다름) |
| 인력 | 투자권유자문인력 상근 1명 이상 |
| 미등록 시 | 3년 이하 징역 또는 1억원 이하 벌금 |

→ 1인 사이드 프로젝트에는 **비현실적**. 앗추는 개별화된 자문을 하지 않으므로 해당하지 않음.

#### B. 유사투자자문업 (중간 규제)

**정의**: **불특정 다수**에게 간행물·통신물·방송 등을 통해 투자조언을 제공하는 영업.

| 항목 | 내용 |
|------|------|
| 방식 | 금융위원회에 **신고** (등록보다 훨씬 간단) |
| 법인 | **개인도 가능** |
| 자본금 | **법적 최소 요건 없음** (사실상 100만원 수준) |
| 인력 | 별도 자격 요건 없음 |
| 절차 | 신고서 작성 + 사전교육 이수 + 제출 |
| 비용 | 직접 하면 거의 0원, 행정사 대행 시 50~100만원 |

→ 앗추가 유료화할 때 가장 **현실적인 경로**.

#### C. 적용 제외 (규제 없음)

**자본시장법 제7조 제3항**: 다음 조건을 **모두** 만족하면 투자자문업으로 보지 않는다.

> 불특정 다수인을 대상으로, **개별성 없는 조언**(개별 투자자를 상정하지 아니하고 다수인을 대상으로 일방적으로 이루어지는 투자에 관한 조언)을, 간행물·출판물·통신물·방송 등을 통해 하는 경우.

**단서 조항 (2024년 개정)**:
> 본문의 조언과 관련하여 온라인상에서 **일정한 대가를 지급한 고객**과 **의견을 교환**할 수 있는 경우에는 그러하지 아니한다.

즉, **무료 + 일방향 = 적용 제외 가능**, **유료 + 양방향 = 투자자문업 해당**.

→ 앗추의 현재 형태(무료, 불특정 다수, 일방향 알림)는 **적용 제외 가능성 높음**.

### 1-3. 유사투자자문업 규제 (2024년 8월 강화)

- 유료 회원 대상 **양방향 채널 전면 금지** (카톡 오픈채팅, 텔레그램 비밀대화 등)
- 허용되는 방식: 일방향 채널만 (텔레그램 공지 채널, 이메일, 웹사이트, 유튜브 등)
- 2024년 금감원 실태점검: 745곳 점검 → **112곳 위반 적발** (전년 대비 2배 증가)
- 2025년 금감원 일제점검: 103곳 위반, 18곳 형사 수사 의뢰
- 5년간 불법행위 5,103건 신고, 수사 의뢰 843건

### 1-4. "도구"인가 "자문"인가 — 핵심 경계선

규제 적용 여부의 핵심은 **서비스가 "투자자문"인지 "데이터 도구"인지**에 달려 있다.

**참고 사례: TradingView** — 이동평균선 알림, 매수/매도 시그널, 백테스트 기능을 유료로 제공하지만 투자자문업 등록을 하지 않았다. 이용약관에 "투자자문가나 자산운용자로 활동하고 있지 않으며, 투자 자문을 제공하는 역할이 아닙니다"라고 명시하고 있다.

| 투자자문 (규제 대상) | 데이터 도구 (규제 대상 아님) |
|---------------------|--------------------------|
| "지금 NVDA를 사세요" | "NVDA가 200일선을 돌파했습니다" |
| "매수 신호입니다" | "기술적 지표가 변했습니다" |
| "이 전략을 따르면 수익 납니다" | "이 데이터는 과거 백테스트 결과입니다" |
| 특정 종목 추천 | 사용자가 직접 필터링/탐색 |
| "상승장을 구매합니다" | "추세 변화를 감지합니다" |

### 1-5. 앗추의 유료화 경로 — 추천 전략

**추천: 유사투자자문업 신고 + 데이터 도구 포지셔닝 병행**

| 경로 | 비용 | 조건 | 앗추 적합도 |
|------|------|------|-----------|
| **A. 데이터 도구로 유료화** | 0원 | "신호/전략" 표현 제거, 면책 강화 | ⭐ 실질적 운영 방식 |
| **B. 유사투자자문업 신고** | 0~100만원 | 개인 가능, 자본금 거의 없음 | ⭐ 법적 안전망 |
| **C. 투자자문업 등록** | 1억원+ | 법인 필수, 전문인력 필수 | ❌ 비현실적 |

**두 가지를 동시에 하는 이유:**
- 신고만 하고 "자문" 느낌으로 운영 → 금감원 점검 대상, 표현마다 규제 적용
- 신고 안 하고 "도구"로만 운영 → 해석이 애매할 때 법적 보호 없음
- **신고 + 도구 포지셔닝** → 신고해둬서 법적으로 깔끔하고, 도구로 운영하니 실질 규제 부담도 적음

**유료화 단계별 실행:**

| 단계 | 시점 | 조치 |
|------|------|------|
| 1 | 지금 (무료) | 표현 순화 + 면책 강화 → 제7조 3항 적용 제외 확보 |
| 2 | 유료화 검토 시 | 금융법 전문 변호사 상담 1회 (30~50만원) |
| 3 | 유료화 결정 시 | 유사투자자문업 신고 (개인, 자본금 거의 없음) |
| 4 | 유료화 실행 시 | 이용약관 + 개인정보 처리방침 + EODHD 라이선스 확보 |

### 1-6. 안전한 운영 방향 (현재)

1. **표현 변경**: "매수/매도 신호" → "기술적 지표 변화", "추세 참고값"
2. **Discord 설정**: 알림 전용 채널 채팅 불가 설정 완료 ✅ (추세 알림·일간 리포트 채널 @everyone 메시지 전송 비활성화)
3. **서비스 정체성**: "투자자문 서비스"가 아닌 "데이터 시각화 도구"로 포지셔닝
4. **면책 강화**: 모든 공개 알림에 "투자 조언이 아닙니다" 추가 — 추세 알림 ✅, 일간 리포트 ✅
5. **무료 유지**: 유료화 전 반드시 변호사 상담 + 유사투자자문업 신고

## 5. 개인정보보호법 — Discord 운영

### 5-1. 현재 수집 현황

| 항목 | 수집 여부 |
|------|---------|
| 사이트 회원가입/로그인 | ❌ 없음 |
| 이메일/개인정보 | ❌ 없음 |
| 쿠키/추적 | ❌ 없음 |
| Discord 사용자 ID/닉네임 | ⚠️ Discord 서버 가입 시 자동 |

### 5-2. Discord 서버의 법적 위치

- 사이트 자체는 회원가입·쿠키·이메일 수집이 전혀 없어 개인정보 처리 없음
- Discord 서버의 사용자 데이터(ID/닉네임)는 **Discord가 수집·저장**하는 것이며, 앗추가 독립적으로 저장·처리하지 않음
- 서버 관리자가 멤버 목록을 열람할 수 있으나, 별도 저장·처리가 없으면 개인정보 처리자로 보기 어려움
- **현 무료·무회원 단계에서는 개인정보 처리방침 실질 의무 없음**

### 5-3. 처리방침이 필요해지는 시점

- 이메일 수집 시작 시
- 회원가입 도입 시
- 유료화로 결제 정보 처리 시

## 6. EODHD 데이터 라이선스

### 6-1. 현재 사용 플랜

- **플랜**: EOD Historical Data — All World ($19.99/mo, Personal use 전용)
- 확인일: 2026-04-06

### 6-2. 라이선스 정책 (EODHD 공식 ToS 원문 기준)

- **허용**: "store, manipulate, and analyze the data for **private, non-commercial purposes**"
- **금지**: "Selling, reselling, retransmitting, redistributing, **displaying**, or granting access to the Information or Services" — **비상업적이어도 public display 명시 금지**
- $399/mo 상업용 Internal Use 플랜조차 외부 표시 불가. 공개 웹사이트 표시는 Enterprise 협의 필요.

### 6-3. 앗추의 실제 구현 방식과 리스크 평가

**앗추의 데이터 처리 방식:**
- 파이프라인(cron) 실행 시에만 EODHD API를 호출 → CSV 파일 생성
- 사용자가 사이트 접속해도 EODHD 서버에 요청 없음 → **EODHD가 사용자 트래픽을 탐지할 수 없음**

| 리스크 유형 | 평가 | 이유 |
|------------|------|------|
| 소송·법적 조치 | 🟢 매우 낮음 | 국제 소송 비용 >> 피해 금액, 소규모 비수익 한국 서비스 |
| 계정 차단 | 🟡 낮음 | 파이프라인 실행 패턴이 정상이면 탐지 어려움 |
| 업그레이드 요청 | 🟡 낮음 | EODHD가 앗추를 직접 발견·인지해야 발생 |
| 유료화 시 리스크 | 🔴 높음 | 수익 발생 순간 상업적 사용 → 라이선스 필수 |

약관상으로는 위반이지만, 실질적 탐지·집행 가능성이 낮아 **현 무료 서비스 단계에서는 감수 가능한 수준**으로 판단. 유료화 시점에는 반드시 재검토 필요.

### 6-4. 조치 사항

1. ✅ 리스크 평가 완료 — 현 단계(무료, CSV 사전 처리 방식)에서 감수 결정
2. ✅ 사이트 푸터에 "데이터 출처: EODHD" 명시 완료 (`AppFooter.jsx`)
3. **유료화 전 필수**: 상업 라이선스 협의 또는 대안 데이터 소스 전환
